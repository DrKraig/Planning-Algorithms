Pre-requisites to run the code:

Python 3 should be installed on your system.

1) Numpy - Install it using 'pip install numpy'

###########################################################
Instructions to run the code:

1) Clone the repository by clicking the big green button located here: https://github.com/SamPusegaonkar/ENPM661
1) Open command prompt or terminal.
2) Navigate to this directory using 'cd ENPM661'
3.1) If OS is windows, type python FinalCode.py
3.2) If OS is Ubuntu, type python3 FinalCode.py
4) Check the generated file in the current directory called 'output.txt'
